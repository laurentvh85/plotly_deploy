# plotly_deploy

Website URL:
https://laurentvh85.github.io/plotly_deploy/

This challenge used Javascript and HTML to analyze data in a JSON file. The purpose of this exercise was to create a website that would dynamically update when an ID was selected. This in turn updated the interactive charts to show us the top 10 bacteria that are found for the individual in bar chart, it shows us the frequency of washing in a gauge chart, and it shows us a scatter graph.
